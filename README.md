# Getting Started with Snowflake Intelligence 日本語

## Overview

Snowflake Intelligenceは、企業が保有する膨大なデータを活用するための強力なソリューションです。分散したデータから必要な情報をすぐに得られないビジネスユーザーの課題や、日々寄せられる問い合わせ対応に忙殺されるデータチームの負担を軽減します。AIエージェントを活用することで、従業員は安全にデータと対話し、より深い洞察を引き出し、必要なアクションを実行できます。これらすべてを、統一された使いやすい一つのインターフェースから実現します。Snowflake Intelligenceは、データと実用的なインサイトの距離を縮め、ビジネスの運営方法そのものを変革します。
